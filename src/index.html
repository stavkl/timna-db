How to Create a Bot Password

Why do I need a bot password?

This form generator uses the Wikibase API to create and update items. A bot password is a separate credential tied to your account that allows automated editing without exposing your main password. Edits will appear normally in the item history under your account name.

Steps to create a bot:

  1. Log in to your Wikibase instance
  2. Go to Special:BotPasswords (add /wiki/Special:BotPasswords to your Wikibase URL)
  3. Click "Create a new bot password"
  4. Enter a bot name (e.g., "FormBot")
  5. Check the following permissions:
    • ✓ High-volume editing
    • ✓ Edit existing pages
    • ✓ Create, edit, and move pages
  6. Click "Create"
  7. Save the generated username (YourName@BotName) and password securely

Note: You only need to create a bot account once. The credentials can be reused for all future sessions.
